# FIAP - Digital Business Enablement - 2º Semestre - NAC1 - Rest API Spring Boot

## Running the app

```bash
BASE_URL =  https://nac-andrey.herokuapp.com/
GET/pacientes
-------------------------
POST/pacientes
JSON body:
 {
    "nome": "name",
    "dataNascimento": "yyyy-mm-dd",
    "cpf": "xxxxxxxxxxx",
    "fumante": boolean
  }
-------------------------
PUT/pacientes/{id}
JSON body:
 {
    "nome": "name",
    "dataNascimento": "yyyy-mm-dd",
    "cpf": "xxxxxxxxxxx",
    "fumante": boolean
  }
-------------------------
DELETE/pacientes/{id} 
```
